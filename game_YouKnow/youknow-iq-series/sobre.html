Sobre o Projeto

YouKnow é um projeto acadêmico desenvolvido como parte da minha formação em Análise e Desenvolvimento de Sistemas (Faculdade) e também como um exercício prático do curso online da B7Web, focado em desenvolvimento web e mobile. O objetivo do jogo é criar uma experiência interativa e divertida para desafiar os usuários com quizzes de várias categorias. A ideia é aplicar na prática os conhecimentos adquiridos durante o curso e na faculdade.

Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias:

  • HTML, CSS e JavaScript: para estruturar e estilizar o site, além de implementar as interações.
  • DALL·E (OpenAI): para a criação de imagens geradas por inteligência artificial que fazem parte do design do jogo.
  • GitHub: para controle de versão e colaboração no código.
  • VS Code: como editor de código.
